EMF-REST

REST-ful APIs out of your EMF models

What is EMF-REST?

EMF-REST is a tool that allows the generation of Truly RESTFul APIs from Ecore models. It enables web developers to generate JSON-based Web APIs for their applications.

EMF-REST complements the existing API generation facilities that the Eclipse Modeling Framework (EMF) already provides, which are limited to Java, and extends them to be used on the Web.

 

REST-ful Services

EMF-REST automatically creates a set of REST services conforming to the JAX-RC specification which you can deploy in your application server. Once deployed, you will have a REST API ready to work with.

JSON Serialization

You API clients can easily interact with your services since all the communication is done via JSON objects, they do not have to realize that they are dealing with models behind your service.

JavaScript API

For those of you thinking of integrating the API in your Web applications, we also generate a JavaScript library which will do all the job to communicate with the generate API.
